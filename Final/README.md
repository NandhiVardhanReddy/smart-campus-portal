# Smart Library Backend

A comprehensive document management and search system built with Flask, MongoDB, Elasticsearch, Redis, and Transformers.

## Features

- **Document Management**: Upload, store, and manage documents
- **Advanced Search**: Keyword, semantic, and hybrid search capabilities
- **Text Extraction**: Automatic text extraction from PDFs, images, and text files
- **Semantic Search**: Vector-based similarity search using sentence transformers
- **Caching**: Redis-based caching for improved performance
- **Scalable Architecture**: Microservices-ready with Docker

## Tech Stack

- **Backend**: Flask
- **Database**: MongoDB
- **Search**: Elasticsearch
- **Cache**: Redis
- **ML/NLP**: Transformers, Sentence Transformers
- **OCR**: Tesseract
- **Containerization**: Docker
